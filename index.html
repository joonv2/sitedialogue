Teaser video: SiteDialogue is to revolutionize the design/engineering/construction industry with cutting-edge AI technology.

Overview

This application is designed to streamline document management and information retrieval on construction sites, addressing the challenge of accessing necessary details without manually searching through documents. It leverages a chatbot interface to facilitate easy communication and information exchange, which may deliver actionable insights from construction documents, decoding various forms of data (i.e., text, voice, image, and video).

Key Features of the Construction Site Document Management and Query System

Document Upload and Analysis

  • Users can upload documents in formats such as PDF, DOC, images, and videos to specific project folders.
  • Automated analysis of uploaded documents by utilizing domain-specific models with LLMs based on the user query.

Query and Response System

  • Interactive chatbot for querying construction project-related information.
  • Search relevant information in the project folders to retrieve necessary details without manually searching through documents.
  • Example: Users ask for installation deadlines and receive answers citing specific documents.
