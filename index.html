SiteDialogue

SiteDialogue is to revolutionize the design/engineering/construction industry with cutting-edge AI technology.

Overview

This application is designed to streamline document management and information retrieval on construction sites, addressing the challenge of accessing necessary details without manually searching through documents. It leverages a chatbot interface to facilitate easy communication and information exchange, which delivers actionable insights from construction documents, containing all forms of data (i.e., text, voice, image, and video), to facilitate real-time communication across project stakeholders.

Key Features of the Construction Site Document Management and Query System

Document Upload and Storage

  • Users can upload documents in formats such as PDF, DOC, and images to specific project folders.
  • Automatic text extraction and sentence-by-sentence embedding vector creation for uploaded documents.
  • Documents are stored with metadata on cloud services like AWS for efficient retrieval and management.

Query and Response System

  • Interactive chatbot for querying construction project-related information like deadlines and specifications.
  • Utilizes large language models to search embedding vectors in the database and fetch relevant document references.
  • Example: Users ask for installation deadlines and receive answers citing specific documents.

Data Improvement and Customization

  • Adaptive learning from user queries to focus on frequently requested information and documents.
  • Potential future features for project-based optimizations tailored to the construction industry.
